# v2.1.0 - Discord Integration

**Дата:** 19.03.2026  
**Статус:** 🟡 Предыдущая версия

---

## 📋 Описание

Версия 2.1.0 представила полноценную интеграцию с Discord: бот с slash-командами, система привязки аккаунтов Minecraft ↔ Discord, и расширенные уведомления через webhooks.

---

## 🆕 Новые функции

### 🤖 Discord Бот

| Команда | Описание | Права |
|---------|----------|--------|
| `/link <ник>` | Привязка аккаунта | Все |
| `/unlink` | Отвязка аккаунта | Все |
| `/verify` | Код верификации | Все |
| `/status` | Статус сервера | Все |
| `/online` | Игроки онлайн | Все |
| `/help` | Справка | Все |
| `/stats` | Статистика AntiBot | Admin |
| `/reload` | Перезагрузка конфига | Admin |
| `/check <игрок>` | Проверка игрока | Admin |
| `/whitelist` | Управление вайтлистом | Admin |

### 🔗 Система привязки аккаунтов

```
Minecraft Nickname ←→ Discord User ID
     ↓                         ↓
   IP Address ←→ Verification Status
```

**Возможности:**
- Автоматическая привязка через команду `/link`
- Защита от множественных привязок с одного IP
- DM верификация с кодом
- Автоматический whitelist для привязанных
- Уведомления в ЛС при входе

### 💬 Webhook уведомления

| Событие | Цвет | Описание |
|---------|------|---------|
| Вход игрока | 🟢 | Логирование входов |
| Выход игрока | ⚫ | Логирование выходов |
| Обнаружение читов | 🔴 | Cheat client detection |
| VPN/Proxy | 🟠 | VPN detection |
| Бот обнаружен | 🟠 | Bot behavior |
| IP заблокирован | 🔴 | IP blocked |
| Кик игрока | 🔴 | Player kicked |
| Режим атаки | 🔴 | Attack mode on/off |
| Подозрительный ник | 🟡 | Suspicious username |

## 🛡️ Модули защиты

### Античиты (ClientBrandDetector)

**Обнаруживает 50+ клиентов:**
```
BeerClient, Wurst, Impact, Aristois, Inertia, Meteor,
RusherHack, Future, Konas, Phobos, SalHack, Kami,
Lambda, LiquidBounce, Sigma, Novoline, Vape, и др.
```

**Паттерны:**
- `*hack*`, `*cheat*`, `*client*`, `*crack*`
- `*bot*`, `*exploit*`, `*inject*`, `*bypass*`

### VPN/Proxy детекция (VPNProxyDetector)

**Интеграция:** proxycheck.io API

**ASN для блокировки:**
```
AS9009, AS20473, AS14061, AS16276, AS24940,
AS60068, AS212238, AS209, AS51167, AS136787, и др.
```

**Типы обнаружения:**
- VPN
- Proxy
- Tor
- Datacenter IP
- Заблокированные диапазоны

### GeoIP (GeoIPChecker)

**Интеграция:** ip-api.com

**Режимы:**
- Белый список стран
- Чёрный список стран
- Кэширование 24 часа

### Анализ поведения (BehaviorAnalyzer)

**Метрики:**
- Паттерны подключений
- Быстрые переподключения
- Использованные ники
- Среднее время сессии
- Подозрительные действия

### Анализ никнеймов (UsernameAnalyzer)

**Проверки:**
- Длина ника (3-16 символов)
- Допустимые символы
- Заблокированные слова
- Паттерны бот-ников
- Энтропия (автогенерированные)

## ⚙️ Конфигурация

### discord-bot
```yaml
discord-bot:
  enabled: true
  bot-token: "TOKEN"
  server-name: "Minecraft"
  account-linking-enabled: true
  dm-verification-enabled: true
  command-whitelist-enabled: false
  allowed-guilds: []
  admin-roles: []
```

### account-linking
```yaml
account-linking:
  auto-whitelist: false
  dm-notification: true
  max-per-discord: 3
  require-verification: false
  trusted-roles: []
```
