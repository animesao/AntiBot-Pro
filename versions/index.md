## 🔄 Миграция между версиями

### v2.1.0 → v2.2.0

Обновите JAR файл. Никаких дополнительных действий не требуется.

### v2.0.0 → v2.1.0

При обновлении с v2.0.0 на v2.1.0:
1. Обновите JAR
2. Перезапустите сервер
3. Настройте Discord бота в `config.yml`
4. Добавьте webhook URL
