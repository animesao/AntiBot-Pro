**Дата:** 14.03.2026

## 📋 Описание

Первый публичный релиз AntiBot Pro. Базовый набор функций для защиты Velocity прокси от ботов и атак.

---

## 🛡️ Базовые функции

### Защита от ботов
- Rate limiting подключений
- Лимит аккаунтов на IP
- Строгий режим (Strict Mode)
- Блокировка по странам (GeoIP)

### Уведомления
- Discord Webhook интеграция
- Логирование событий
- Настраиваемые цвета

### Конфигурация
- YAML конфигурация
- Белые списки IP/игроков
- Настраиваемые сообщения
