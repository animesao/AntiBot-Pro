# v2.3.0 - Re-verification & Update Checker

**Дата:** 22.03.2026  
**Статус:** ✅ Текущая версия

---

## 📋 Описание

Версия 2.3.0 добавляет полноценную систему повторной верификации и автоматическую проверку обновлений через GitHub.

---

## 🆕 Новые функции

### 🔐 Система повторной верификации

| Настройка | Описание | По умолчанию |
|-----------|----------|--------------|
| `reverify-on-ip-change` | Верификация при смене IP | `true` |
| `reverify-periodic` | Периодическая верификация | `true` |
| `reverify-period-hours` | Период проверки (часы) | `24` |

### 🌐 GitHub Update Checker

- Автоматическая проверка обновлений при запуске
- Уведомление в консоли при наличии новой версии
- Команда `/antibot update` для ручной проверки
- Команда `/update` в Discord боте

### 📱 Новые команды

| Команда | Описание | Права |
|---------|---------|--------|
| `/antibot verify <код>` | Верификация 2FA в игре | Все |
| `/antibot reverify` | Запросить новый код | Все |
| `/antibot update` | Проверить обновления | Admin |
| `/update` (Discord) | Проверить обновления | Все |

---

## 🔧 Настройка верификации

```yaml
account-linking:
  # Требовать верификацию
  require-verification: true
  
  # Верификация при смене IP
  reverify-on-ip-change: true
  
  # Периодическая верификация
  reverify-periodic: true
  reverify-period-hours: 24
```

---

## 🚀 Как работает верификация

### При смене IP:
```
1. Игрок меняет IP
2. При входе система видит несовпадение
3. Отклоняет вход с сообщением
4. Игрок получает код через /verify в Discord
5. Вводит /antibot verify <код>
6. Успешный вход
```

### Периодическая:
```
1. Прошло 24 часа с последней верификации
2. При входе система видит просроченную верификацию
3. Отклоняет вход
4. Процесс тот же что и при смене IP
```
