## 🔐 Безопасность

### Исправленные уязвимости

1. **Memory Leak** - DiscordBot reload создавал новый инстанс без удаления старого
2. **HTTP → HTTPS** - Небезопасные соединения с API сервисами
3. **Thread Safety** - ExecutorService без daemon потоков и awaitTermination

### Улучшения

- Добавлены shutdown hooks для всех async компонентов
- Корректное завершение потоков при выключении сервера
- Gson парсинг вместо хрупких regex-based парсеров
