## [v2.3.0](versions/v2.3.0.md) - 22.03.2026

### 🆕 Новые функции

| Функция | Описание |
|---------|----------|
| **2FA Верификация** | Привязанные аккаунты должны верифицироваться через Discord или /antibot verify |
| **GitHub Release Checker** | Автоматическая проверка обновлений из GitHub |
| **IP Change Detection** | Повторная верификация при смене IP |
| **Periodic Re-verification** | Периодическая верификация по таймеру |
| **/antibot reverify** | Запрос нового кода верификации |
| **/antibot update** | Проверка обновлений |
| **/update в Discord** | Проверка обновлений через Discord |

### ⚙️ Новые настройки

```yaml
account-linking:
  reverify-on-ip-change: true     # Верификация при смене IP
  reverify-periodic: true         # Периодическая верификация
  reverify-period-hours: 24       # Период в часах
```

## [v2.2.0](versions/v2.2.0.md) - 22.03.2026

### 🛠️ Исправления и улучшения

| Улучшение | Описание |
|-----------|----------|
| **Gson JSON** | Заменён кастомный JSON парсер на Gson для надёжности |
| **Shutdown Hooks** | Корректное завершение работы Discord компонентов |
| **Memory Leak Fix** | Исправлена утечка памяти при reload Discord бота |
| **HTTPS** | Все API запросы теперь используют HTTPS |
| **Thread Safety** | Улучшена потокобезопасность ExecutorService |
| **Code Quality** | Удалены дубликаты, улучшено качество кода |

### 📊 Изменения в коде

- `AccountLinkManager.java`: 475 → 260 строк (удалён кастомный парсер)
- `DiscordWebhook.java`: 589 → 320 строк (удалён кастомный парсер)
